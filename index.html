🎯 Justificación Breve

La huerta escolar automatizada busca la alimentación saludable y el aprendizaje práctico de tecnologías. Mediante sensores se podrá monitorear la humedad del suelo, la luz solar y la temperatura, garantizando mejores condiciones para el crecimiento de las plantas.

🔧 Posibles Sensores a Emplear

  • Sensor de humedad del suelo: mide el nivel de humedad para saber cuándo regar.
  • Sensor de temperatura y humedad (DHT11/DHT22): registra condiciones climáticas.
  • Sensor de luz (LDR o BH1750): detecta luminosidad para evaluar exposición solar.

⚙️ Explicación del Diseño Técnico

  • Los sensores recogen datos en tiempo real.
  • El ESP32 procesa la información y decide si activar el riego automático.
  • Los datos se envían a la app en App Inventor mediante Bluetooth.
  • El usuario puede visualizar y controlar manualmente el sistema de riego.

🔗 Justificación de Comunicación y Aplicación

Comunicación: Bluetooth → No requiere Internet, bajo consumo, conexión rápida.

Aplicación: App Inventor → Fácil, accesible, integra módulos Bluetooth.
